How to play

  • Use the available tools to try to match the image on the right. Note that if the canvas is not full-size, you should be comparing only with the top-left portion that matches, not attempting to draw the entire image re-scaled.
  • Checks are sent with each percentage point of similarity, which can be seen at the bottom right along with the maximum score in logic.
  • The Archipelago Text Client is accessible under the Extras menu.

Additional features

  • Use File->Open Goal Image or Ctrl+O to use a custom image. The image will be stretched to fill the canvas and if transparency is present, will overlay the default image.
  • Use Extras->Similarity Mode or Ctrl+Shift+M to visualize the similarity between your image and the target. White is perfect similarity, black is no similarity.
  • Use Extras->Difference Mode or Ctrl+M to visualize the differences between your image and the target. If a given color is not gray, you should aim to shift the hue toward the shown color. If the given color is especially light or dark, you should lighten or darken to match. A perfectly similar color will appear the same neutral gray as the area around the canvases.
  • The Pick Color tool, once unlocked, can be used on the target image as well as on the main canvas.
